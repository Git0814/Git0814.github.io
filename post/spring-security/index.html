Spring Security 快速开始

2022-11-19 2 min read

Spring Security 是一个功能强大且高度可定制的身份验证和访问控制框架。它是保护基于 Spring 的应用程序的事实标准。

Spring Security 是一个专注于为 Java 应用程序提供身份验证和授权的框架。与所有 Spring 项目一样，Spring Security 的真正强大之处在于它可以轻松扩展以满足自定义需求

Spring Security快速开始：
在一个springboot项目中引入security依赖：

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>

写一个测试接口：

@RestController
public class Security {

    @GetMapping("/hello")
    public String getmessage(){
        return "hello security";
    }
}

启动项目，访问localhost:8080/hello

访问后可以看出并不是像传统的springboot项目一样直接跳转到hello页面，而是先跳转到一个login页面
此时使用默认用户名user 以及项目启动时随机生成的字符串密码进行登陆

登陆后成功跳转到hello页面
